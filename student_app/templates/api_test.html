Student Management API

RESTful API Documentation & Testing Interface

API Key Required: Include this header in all requests (except /api/docs)
X-API-Key: your-secret-api-key-123

GET Get All Students

Retrieve a list of all students

GET http://localhost:5001/api/students

GET Get Single Student

Retrieve a specific student by ID

GET http://localhost:5001/api/students/{id}

POST Create Student

Add a new student to the database

POST http://localhost:5001/api/students

PUT Update Student

Update an existing student's information

PUT http://localhost:5001/api/students/{id}

DELETE Delete Student

Remove a student from the database

DELETE http://localhost:5001/api/students/{id}

GET Get Statistics

Get statistics about students

GET http://localhost:5001/api/stats
